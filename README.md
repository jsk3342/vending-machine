## 목표

피그마 툴을 이용하여 퍼블리셔와 협업하여 정확히 구현하는 능력을 배양하고 자바스크립트를 활용하여 자판기의 기능을 구현하는것이 목표 입니다.

## 어려웠던 점

### 1. 레이아웃 잡기

float를 이용하여 구성

### 2. 결제 금액 텍스트 정렬

span 태그를 이용하여 구현 하였으며, 인라인 요소로 패딩으로 조절 하였습니다.

## 배운점

### border-box

마진, 패딩값을 주면서 레이아웃이 조금씩 엇나갔는데 border-box 이용하여 사이즈 조절하였습니다.
